# App Instructions

This repository contains a frontend app. Below are common scripts and usage notes.

## Scripts

- `npm start`: Run the app in development at `http://localhost:3000`.
- `npm test`: Launch the test runner in watch mode.
- `npm run build`: Create a production build in the `build` folder.

## Notes

- Built with Create React App; see CRA docs for advanced configuration and troubleshooting.
- Keep environment variables in `.env` files; these are ignored by Git.
